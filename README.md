# MAURF-dVRK

Multi-Agent Unified Reinforcement Framework for da Vinci Research Kit

![Surgical Scene](assets/images/surgicalScene.png)

## Project Website

Visit our project website: [https://luigimuratore.github.io/MAURF-dVRK/](https://luigimuratore.github.io/MAURF-dVRK/)

## About

MAURF-dVRK is a comprehensive framework for multi-agent reinforcement learning applied to surgical robotics using the da Vinci Research Kit. Our framework enables autonomous coordination between multiple agents for complex surgical tasks.

## Features

- Multi-agent coordination and collaboration
- Unified reinforcement learning framework
- Graphics User Interface (GUI)
- ROS integration
- Integration with da Vinci Research Kit
